# PF1 development
this is a development for our project.

# Requirement
* ros melodic
# Installation
```
For ros melodic install
$ chmod +x install_ros_melodic.sh
$ ./install_ros_melodic.sh
```

```
For requirement package
$ chmod +x install.sh
$ ./install.sh
```
# Usage

```
$ source ~/catkin_ws/devel/setup.bash
$ roslaunch PF1 PF1.launch
```
